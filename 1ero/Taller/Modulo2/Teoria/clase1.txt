calculos aritmeticos

int i = 4/3; => división entera i = 1
double d1 = 4.0/3.0 => división real d1 = 1.33333
double d2 = 4/3 => división entera d2 = 1.0
double d3 = (double) 4/3 => división real d3 = 1.33333 => de esta manera se hace una aclaración de como quiere que se retorne
